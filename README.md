# BEng-Project
Final year project for BEng Robotics examining whether machine vision could be employed to aid in manufactoring by attempting to track a piece of cargo as it moved through a model factory.  
  
This folder contains both the MATLAB code files used as well as all the data (image and video files) used. Also enclosed are pdfs of the dissertation, interim report and poster created as part of the project.
